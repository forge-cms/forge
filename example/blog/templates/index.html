What's included

  • Posts list

    Published devlog entries with dates, tags, and excerpts. HTML rendered via forge:head.

  • RSS Feed

    RSS 2.0 feed for all published posts. Regenerates automatically on publish.

  • Aggregate feed

    Top-level /feed.xml merging all module feeds (in a multi-module app).

  • Sitemap index

    Sitemap index linking to /posts/sitemap.xml. Empty until first publish event.

  • llms.txt

    Compact AI index: one line per published post for LLM context windows.

  • llms-full.txt

    Full markdown corpus — all post bodies concatenated for AI assistants.

  • robots.txt

    Auto-generated with sitemap link. AskFirst AI crawler policy enabled.

  • Redirects manifest

    Live JSON manifest of all registered redirect rules.

⏱ One post is Scheduled and will publish automatically in ~2 minutes. The sitemap at /posts/sitemap.xml updates when it does.
